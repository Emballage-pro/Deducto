Deducto is a Python open-source intelligence (OSINT) tool that automatically extracts basic information from LinkedIn profiles using the Brave Search API.
It is designed for security researchers, intelligence professionals, and anyone with a legitimate need to collect publicly available information.
Features :
Brave Search API — leverages the Brave search engine to find targeted LinkedIn profiles
Basic profile extraction — name, job title, company, location, and other public data
Simple and efficient — lightweight command-line interface
